# pair list

List all available co-authors

## Synopsis

List all configured co-authors from your configuration file.
		This command displays each co-author's name, email, and alias
		in a formatted table.

```shell
pair list [flags]
```

## Examples

```shell
# List all co-authors
pair list
```

## Options

```text
  -h, --help   help for list
```
