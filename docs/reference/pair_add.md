# pair add

Add one or more co-authors to Git commits by alias or index

```shell
pair add [alias or index]... [flags]
```

## Examples

```shell
pair add jane john
```

## Options

```text
  -h, --help   help for add
```

## Options inherited from parent commands

```text
  -c, --config string   config file path (default "/Users/philipp.eckel/.pair.json")
```
